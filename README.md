# Clone project
```bash
git init
git clone https://github.com/truonganhvu205/little-lemon-django-app.git
cd little-lemon-django-app
```

## Create venv & activate venv
```bash
python3 -m venv env
source env/bin/activate
```

## Install Django
```bash
pip3 install django
```

# Run server
```bash
python3 manage.py runserver
```

## Deactivate venv
```bash
deactivate OR exit
```
