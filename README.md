# Preparing
## Create venv
```bash
python3 -m venv env
```

## Activate venv
```bash
source env/bin/activate
```

## Install Django
```bash
pip3 install django
```

## Create Django project
### Create Little Lemon
```bash
django-admin startproject littlelemon .
```

### Create restaurant
```bash
python3 -m django startapp restaurant
```

# Getting Started
```bash
python3 manage.py runserver
```
